# Paywall Feature

The Paywall feature manages subscriptions and entitlements.

## Explanation
The paywall defines the boundary between free and premium experiences.  
It presents upgrade screens and ensures features like 10-card decks, Reports, and premium messaging are gated appropriately.  
RevenueCat integration handles subscriptions, free trials, and entitlement checks across iOS and Android.

## What it does
- Presents upgrade screens with plan options
- Unlocks premium features (10-card deck, Reports, email/Slack send)
- Provides entitlement checks to gate access
- Supports trials, monthly, and annual subscriptions
