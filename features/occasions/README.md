# Occasions Feature

The Occasions feature highlights birthdays, anniversaries, and other key dates.

## Explanation
Occasions surface important personal milestones to strengthen relationships.  
The system ingests dates from Google and Microsoft accounts for premium users and creates reminders in the app.  
Users can view upcoming occasions and take action directly from the list.

## What it does
- Tracks birthdays, work anniversaries, and similar milestones
- Displays an upcoming occasions list
- Encourages timely outreach for special dates
- Available only to premium users
