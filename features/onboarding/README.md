# Onboarding Feature

The Onboarding feature introduces new users to the app and sets up initial permissions.

## Explanation
Onboarding ensures users can start using the app quickly and effectively.  
It walks through granting contact permissions, setting tag and cadence presets, connecting external sources, and enabling notifications.  
This flow creates a strong first impression and sets the foundation for daily use.

## What it does
- Guides users through granting contact permissions
- Lets users set quick tag and cadence presets
- Offers premium users the option to connect Google, Microsoft, or Slack
- Prompts for enabling notifications
