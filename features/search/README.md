# Search Feature

The Search feature enables global lookups for contacts and notes.

## Explanation
Search makes it easy to find any person or context in the app.  
Users can search by name, tag, or note content, and quickly jump into a contact or note.  
It supports multi-contact linking by allowing notes to be attached to several people directly from search results.

## What it does
- Provides global search across contacts and notes
- Supports filtering by tags and text content
- Enables quick navigation to details
- Allows multi-contact note attachment
