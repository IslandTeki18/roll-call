# Sources Feature

The Sources feature manages external integrations for contact syncing.

## Explanation
Sources expand the contact base beyond the device.  
Premium users can connect Google, Microsoft, and Slack to import and sync contacts.  
The feature manages authentication, permission scopes, and connection health, ensuring external data integrates cleanly with the core contact model.

## What it does
- Connects to external sources (Google, Microsoft, Slack)
- Manages OAuth flows and permission scopes
- Syncs and normalizes external contact data
- Tracks provider connection health
